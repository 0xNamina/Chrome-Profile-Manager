# Chrome Profile Manager

Tool Python untuk mengelola banyak profil Chrome dengan mudah.

## Cara Pakai
1. Pastikan sudah install Python 3.7+ dan Google Chrome
2. Jalankan script: python main.py
3. Pilih mode (Pemula/Advanced)
4. Ikuti menu di program

## Fitur
- Buat profil Chrome baru otomatis
- Kelola akun dalam folder
- Buka beberapa akun sekaligus
- File global auto-launch

## Contoh Command Chrome

chrome.exe --profile-directory="Profile 1"

## File
- `main.py` - Program utama
- `data.json` - Data akun (auto-generated)
